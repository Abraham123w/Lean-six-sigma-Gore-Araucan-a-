4.2. Medir

Recolección de datos, Minería de Texto y establecimiento de Línea Base

Metodología de Extracción de Datos

En esta etapa se recopiló información del Anexo 7 (comentarios de evaluadores sobre no aprobación técnica). La estructuración y limpieza se realizó mediante Python para agilizar el proceso. Utilizando técnicas de minería de texto, se extrajeron comentarios y se almacenaron en formato JSON para identificar patrones recurrentes sin modificar el contexto original.
